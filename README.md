# End-to-End CI/CD Pipeline using Jenkins, Docker & AWS EKS

## Project Overview

This project demonstrates an automated CI/CD pipeline for deploying a Java Maven web application on Amazon EKS using Jenkins, Docker, Amazon ECR, and Kubernetes.

---

## Technologies Used

- Linux
- Git
- GitHub
- Maven
- Jenkins
- Docker
- Amazon ECR
- Amazon EKS
- Kubernetes
- AWS Load Balancer

---

## CI/CD Workflow

GitHub
↓

Jenkins

↓

Maven Build

↓

Docker Image

↓

Amazon ECR

↓

Amazon EKS

↓

AWS Load Balancer

---

## Features

- Automated Build
- Automated Deployment
- Docker Containerization
- Kubernetes Deployment
- Rolling Updates
- Load Balancer Exposure

---

## Project Outcome

Successfully automated the complete deployment pipeline using Jenkins, Docker, Amazon ECR, and Amazon EKS, reducing manual deployment effort and enabling scalable application deployment.
